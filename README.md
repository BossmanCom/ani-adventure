# Ani Adventure

Click-based 2D adventure game starring Ani the foxgirl.

## Current Progress
- Character sprite being prepared
- Starting with small wooden house scene
- Click-to-move planned

## Tech Stack
- HTML5 Canvas
- Vanilla JavaScript
- Pixel / 2D anime style

## Repo Structure (planned)
- `/assets` - sprites and images
- `/js` - game code
- `index.html` - main entry
